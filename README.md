# SmartShield IoT 🏠🔐

SmartShield IoT is an ESP32-based smart home automation and security system that enables real-time monitoring and control of home devices using the Blynk IoT mobile application.

---

## 🚀 Features

* 🔐 Smart Door Lock Control (Remote Access)
* 🌡️ Temperature Monitoring
* 💧 Humidity Monitoring
* 📱 Mobile Control via Blynk IoT App
* ☁️ Real-time Data Updates
* 🔔 Instant Alerts & Notifications

---

## 🧠 Project Overview

This project uses the ESP32 microcontroller to create a smart home system that allows users to monitor environmental conditions and control a door lock remotely.

All controls and data are accessible through the Blynk IoT mobile app, providing a seamless and user-friendly experience.

---

## 🧰 Tech Stack

* **Microcontroller:** ESP32
* **Sensors:** DHT11 / DHT22 (Temperature & Humidity)
* **IoT Platform:** Blynk IoT
* **Programming Language:** C++ (Arduino IDE)
* **Mobile App:** Blynk IoT App

---

## 📲 Blynk IoT Integration

* Create a project on Blynk IoT
* Add widgets for:

  * Temperature display
  * Humidity display
  * Door lock control button
* Connect ESP32 using Blynk Auth Token
* Monitor and control everything from your phone

---

## ⚙️ Hardware Components

* ESP32
* DHT11/DHT22 Sensor
* Relay Module / Servo Motor (for door lock)
* Jumper Wires
* Power Supply

---

## 🔌 Working Principle

1. ESP32 reads temperature and humidity from the sensor
2. Data is sent to the Blynk cloud
3. User monitors data via mobile app
4. User can control door lock remotely
5. ESP32 executes commands instantly

## ▶️ How to Run

1. Install Arduino IDE
2. Install ESP32 board package
3. Install required libraries:

   * Blynk
   * DHT sensor library
4. Add your:

   * WiFi credentials
   * Blynk Auth Token
5. Upload code to ESP32
6. Open Blynk app and monitor/control
